Validate Location for Stock Picking (Odoo 17)

"No more Mistakes on Stock Pickings"

This module allows the source and destination location of a picking operation to be checked. This gives more awareness to employees working in multiple warehouses/locations.

Core Features:

  • Extra layer of validation: Makes the user aware of the source and destination locations.
  • Overriding button logic: This module overrides the action of the button and does not interfere with the logic of button_validate().

Custom logic on top of the "Validate" button.

Double check the source and destination location of the product.
